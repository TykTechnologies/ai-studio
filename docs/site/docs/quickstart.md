# Quickstart

The fastest way to get started with Tyk AI Studio is using Docker Compose.

See the **[Docker Compose Deployment](./deployment-docker.md)** guide for complete instructions covering:

- AI Studio standalone setup (with embedded gateway)
- AI Studio + Microgateway (hub-spoke architecture)
- PostgreSQL database configuration
- Analytics and monitoring

## Other Deployment Options

- **[Kubernetes / Helm](./deployment-helm-k8s.md)** — Production deployment on Kubernetes with Helm charts
- **[Bare Metal / VM](./deployment-packages.md)** — Install via DEB/RPM packages on Linux servers

## After Deployment

Once deployed, proceed to the [Initial Configuration](./configuration.md) guide to set up your first LLM, users, and applications.
